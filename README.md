# StudyTrack AI – Student Study Habit Recommender 📚🤖

## Project Overview

StudyTrack AI is a data-driven dashboard that analyzes students' study habits and academic performance.
The system uses machine learning techniques to identify study patterns, predict student scores, and provide personalized recommendations for improvement.

## Features

* 📊 Student performance dashboard
* 📈 Visual analysis of academic scores
* 🤖 AI-based clustering of study habits
* 🎯 Score prediction using machine learning
* 🚨 Academic risk detection
* 💡 Personalized study recommendations

## Technologies Used

* Python
* Streamlit
* Pandas
* Scikit-learn
* Matplotlib

## Dataset

The project uses a student performance dataset containing information such as:

* Study hours per week
* Attendance percentage
* Sleep hours
* Stress level
* Academic scores

## How to Run the Project

1. Install required libraries:
   pip install streamlit pandas matplotlib scikit-learn

2. Run the Streamlit app:
   streamlit run app.py

## Project Goal

The goal of this project is to help students improve their academic performance by analyzing study habits and providing intelligent recommendations.
